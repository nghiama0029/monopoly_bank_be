# Learnings – Kiến Thức Tích Lũy Của Dự Án

Thư mục này chứa các file được agent tự động cập nhật trong quá trình làm việc với dự án.
Mỗi file tập trung vào một khía cạnh cụ thể.

## Cấu Trúc

| File | Nội Dung |
|------|----------|
| `architectural-decisions.md` | Quyết định kiến trúc đã được đội thông qua |
| `coding-conventions.md` | Convention đặt tên, cấu trúc code của dự án |
| `common-mistakes.md` | Lỗi hay gặp cần tránh |
| `dependency-versions.md` | Phiên bản thư viện đang dùng |
| `domain-glossary.md` | Thuật ngữ nghiệp vụ |

## Cách Sử Dụng

Agent sẽ tự động đọc thư mục này theo rule `load-learnings.md` trước mỗi tác vụ.
Developer có thể chỉnh sửa trực tiếp các file này nếu cần.
